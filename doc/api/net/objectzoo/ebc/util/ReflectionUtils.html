public static Class<?> getRawGenericSuperTypeArgument(Class<?> concreteGenericType,

Retrieve the raw type of the generic super class' type argument. This will only work correctly when the given concrete generic type specifies concrete classes for the type parameters of it's super class. For type variables or wildcard type the raw type of the upper bound will be returned.

Parameters:
concreteGenericType - the class of the implementation of a generic type
numberOfArgument - the number of the generic super class' type argument
Returns:
the raw type of the generic super class' type argument
Throws:
IllegalArgumentException - if no raw type can be determined or the super class is not a parameterized type
