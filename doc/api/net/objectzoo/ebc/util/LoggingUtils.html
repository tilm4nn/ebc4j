log

public static void log(Logger logger,
                       Level level,
                       String message,
                       Object parameter)
This method logs the given message and a String representation of the given parameter obtained by String.valueOf(Object). This is provided for logging as a workaround so that this library can be used in GWT projects because GWT's Logger implementations does not provide the method Logger.log(Level, String, Object).

Parameters:
logger - the Logger to log to
level - the Level to use
message - the message to log
parameter - the parameter to log
