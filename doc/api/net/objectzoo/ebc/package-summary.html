Package net.objectzoo.ebc

This package contains interfaces that can be used to define EBCs.

See:
          Description

Interface Summary
CanProcess<ProcessParameter> An EBC that CanProcess is capable of processing input and thus provides an Action to do so.
CanStart An EBC that CanStart can be invoked to perform same work and thus provides an Action0 to do so.
Configurable EBCs that are Configurable can be configured with an arbitrary number of parameters.
DependsOn<Dependency> An EBC that DependsOn another object, can be explicitly tagged with this interface.
EntryPoint This is a special marker interface that can be used to tag an EBC as the EntryPoint of an application.
SendsResult<ResultParameter> An EBC that SendsResult is capable of sending the output of it's work to subscribed listeners and thus provides an Event to do so.
SendsSignal An EBC that SendsSignal is capable of sending a signal when it's work completes to subscribed listeners and thus provides an Event0 to do so.
