public abstract class JoinToConstructableObject<Input1,Input2,Output,OutputElement>
extends JoinBase<Input1,Input2,Output>

This is a base class that can be used to implement Join EBCs that create their output elements by calling a constructor that takes the two input values as parameters.

Constructor Detail

JoinToConstructableObject

protected JoinToConstructableObject()
Initializes this JoinToConstructableObject with a constructor for the output elements determined from the OutputElement type parameter by taking a constructor that has the fitting parameter types for this Join's input types. The types of the generic type parameters are used to determine a fitting constructor using Class.getGenericSuperclass(). This approach will only work if the super class has defined type parameters. This must be ensured by making all subclasses that do not specify concrete classes for the type parameters abstract.

Throws:
IllegalArgumentException - if the output element type does not have a fitting constructor
