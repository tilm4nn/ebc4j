public class MockAction<T>
extends Object
implements net.objectzoo.delegates.Action<T>

The MockAction is an Action implementation that can be used in (unit) tests to collect the results of a tested EBC. The MockAction can be invoked as many times as the maximum invocations allowed value given at construction time before throwing an AssertionError.
