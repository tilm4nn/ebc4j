public class ProcessAndClearThreadLocalStateStore<T>
extends ProcessAndResultBase<T,T>

This class implements a decorator EBC that simply forwards the processing to the next component in the flow and clears the ThreadLocalStateStore after the flows execution. This only works for the current synchronously executing part of the flow and thus must be introduced in the beginning of every part of the flow that is executed asynchronously.
